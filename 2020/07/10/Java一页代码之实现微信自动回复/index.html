前言

现在，它还可以替我们聊天，是的，它说：Yes，I do!

可喜可贺的是，帮我回复女朋友微信消息的智能程序写好了，女朋友没了。

所以屏幕前好奇这个程序的你，还看什么代码，赶紧滚去找对象。

有对象了？那我劝你且行且珍惜。本程序对运行后果概不负责。
